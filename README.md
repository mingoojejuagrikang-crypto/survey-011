## 변경 내역 (Changelog)

> 최근 변경이 맨 위입니다. 어떤 점이 바뀌었는지 쉬운 말로 정리합니다.

- **v0.12.0** (2026-06-17) — 실기기 테스트(v0.11.0, 비 오는 비닐하우스) 로그 분석과 사용 요청을 반영했습니다.
  - **작동하지 않던 출력 전환 버튼을 없애고, 그 자리에 "현재 입력 장치"를 보여줍니다(입력탭).** 눌러도 글자만 바뀌고 실제로는 안 바뀌던 스피커/이어폰 버튼을 뺐습니다(음성 입력 방식은 휴대폰 설정에서 바꾸세요). 대신 지금 어떤 마이크로 듣고 있는지를 **📱 내장 마이크 / 🎧 블루투스 / 🎧 유선 이어폰**으로 표시합니다.
  - **이상치 알람 팝업에 "어느 샘플·언제 값"인지 함께 보여줍니다(입력탭).** 알람을 확인할 때 샘플을 헷갈리지 않도록, 항목 이름과 **샘플(또는 행 번호)**, 그리고 직전 값이 **언제(예: 2026-05-13) 조사한 값**인지 날짜를 함께 띄웁니다.
  - **이상치 알람 팝업 색을 주황에서 빨강으로 바꿨습니다(입력탭).** 더 눈에 잘 띄게 했습니다. 증가/감소 방향은 글자로 그대로 안내합니다.
  - **"수정" 할 때 어떤 항목을 고치는지 표시하고, 그 칸을 강조합니다(입력탭).** "수정"이라고 말하면 **"수정 값: ○○"**를 파란색으로 띄우고 해당 항목 칸을 밝게 표시한 뒤, 새 값을 기다립니다.
  - **여러 세션을 CSV로 내보낼 때 세션마다 따로 파일로 묶어 줍니다(데이터탭).** 예전엔 여러 세션을 한 파일에 칼럼을 겹쳐 담아 보기 불편했는데, 이제 **세션별 CSV를 ZIP 한 개**로 모아 내려받습니다(한 세션만 고르면 그냥 .csv 한 개).
  - **자동입력 항목을 여러 값에서 한 값으로 바꾸면 음성확인이 자동으로 꺼집니다(설정탭).** 값이 더는 바뀌지 않는데도 음성확인이 켜진 채 남던 문제를 고쳤습니다(필요하면 직접 다시 켤 수 있습니다).
  - **"로그 ZIP" 이름을 "사용자 로그"로 바꿨습니다(데이터탭).** 기능은 그대로입니다.
  - **"비교" 탭을 잠시 숨겼습니다.** 다른 탭을 안정화한 뒤 더 보기 좋게 새로 만들 예정입니다(설정·기록은 그대로 보존되어 다시 켤 수 있습니다).
- **v0.11.0** (2026-06-17) — 스피커폰(기기 스피커) 사용 시 오인식을 줄였습니다.
  - **안내 음성이 끝난 직후의 잔향을 잠깐 무시합니다(스피커폰 모드).** 스피커로 들을 때는 기기에서 나온 안내 음성이 마이크로 되돌아와 엉뚱한 값이나 명령으로 잘못 인식되곤 했습니다. 이제 안내가 끝난 뒤 아주 짧은 순간(0.25초) 동안의 입력을 막아 그 되먹임을 걸러냅니다. 이어폰을 쓸 때는 종전처럼 안내 도중에도 바로 말할 수 있어 영향이 없습니다.
- **v0.10.0** (2026-06-16) — 실기기 테스트(v0.9.0) 로그 분석과 사용 요청을 반영했습니다.
  - **"비교" 탭을 새로 설계했습니다.** 보고 싶은 조건(농가·처리 등)을 여러 개 골라 **동시에 만족하는 것만** 추려 보고, 비교할 **기준 회차와 "몇 회차 전"을 직접** 정합니다. 표에서 **보고 싶은 샘플 행만 골라** 보고, 어떤 항목을 앞 열(묶음)에 둘지·어떤 측정값을 볼지 토글로 바꿉니다. 여러 샘플을 더하거나 평균 내지 않고 **샘플마다 한 줄로** 그대로 보여줍니다.
  - **소수점을 못 알아들으면 값을 추측하지 않고 다시 묻습니다(입력탭).** 예전엔 "111 점 에"처럼 소수점 뒷자리가 흐릿하면 임의로 채워 조용히 저장될 수 있었는데, 이제 **정수 부분(111)은 그대로 두고 "소수점 아래만"** 다시 말해 달라고 물어 잘못된 값 저장을 막습니다.
  - **이상치 알람 팝업을 화면 한가운데로 옮기고 크게 키웠습니다(입력탭).** 위쪽에 작게 떠 놓치기 쉽던 알람을 **화면 중앙에 1.3배** 크기로 띄우고, 화면을 넘지 않도록 크기를 맞춰 보여줍니다.
- **v0.9.0** (2026-06-16) — 실기기 테스트(v0.8.0) 로그 분석과 사용 요청 5가지를 반영했습니다.
  - **이상치 알람 때 이전값과 현재값을 화면에 띄웁니다(입력탭).** 음성 안내만으로는 스쳐 지나가기 쉬워서, 알람이 울리면 "직전 ○○ → 현재 ○○"를 화면 위쪽에 보여줍니다. 또 **"증가/감소" 알람은 차이를 절대값으로**(예: "직전보다 20.5 증가했습니다"), **변동률 알람은 기존처럼 %로**(예: "21.0% 증가") 말합니다.
  - **입력 중에도 스피커/이어폰을 바꿀 수 있는 버튼이 생겼습니다(입력탭 위쪽).** 휴대폰 설정을 열지 않고 바로 전환해 볼 수 있습니다. (스피커로의 강제 전환은 기기·iOS 사정에 따라 실제로 바뀌지 않을 수도 있어, 이번엔 실험적으로 넣고 실기기에서 효과를 확인합니다.)
  - **순차·복수선택 자동입력 항목은 음성확인이 자동으로 켜집니다(설정탭).** 자동으로 채워지더라도 행마다 값이 바뀌는 항목(예: 조사나무 1·2·3…)은 바뀐 값을 음성으로 들려주는 게 맞아서, 그 방식으로 바꾸면 음성확인이 "유"로 켜집니다. 들을 필요가 없으면 직접 "무"로 끄면 그대로 유지됩니다.
  - **음성 클립의 긴 공백을 잘라냅니다(데이터탭).** 값을 말하기 전 한참 뜸을 들였을 때 클립 가운데에 길게 남던 침묵을 줄여, 말한 부분만 짧게 담깁니다(앞뒤 약간의 여유는 남깁니다).
  - **"조회" 탭을 **"비교"**로 이름을 바꾸고 한 장의 표로 정리했습니다.** 카드·피봇·그룹·퍼센트 보기 선택을 없애고, 공통 항목(농가명·라벨 등)과 조사 회차를 위에 고정한 뒤, 샘플(처리·조사나무·조사과실)을 앞 열에, 측정값(횡경·종경)의 **직전/현재를 뒷 열**에 나란히 둡니다. 조사 기간은 **"2026. 24주차 (06-08 ~ 06-14)"** 형식으로 보여줍니다.
  - **(실험) "빠른 인식" 켜기가 설정탭에 생겼습니다.** 말을 멈춘 뒤 인식이 확정될 때까지의 짧은 기다림을 줄입니다. 소수점을 늦게 말하면 잘릴 수 있어 기본은 꺼져 있습니다.
- **v0.8.0** (2026-06-15) — "추세 확인"을 **"이상치 알람"**으로 바꾸고, "조회" 탭을 휴대폰 세로 화면에 맞게 다시 설계했습니다.
  - **"이상치 알람"으로 바뀌었습니다(설정탭).** 이전의 "추세"는 사라지고, 항목별로 **"증가"** 또는 **"감소"**를 정해 두면 입력한 값이 직전 조사보다 그 방향으로 변할 때 "이상치"로 알려줍니다. 즉 평소와 다른 값이 들어오면 바로 확인을 요청합니다. (예전 "커져야 정상"과는 **반대** 개념입니다.)
  - **변동률(%) 알람이 추가됐습니다.** 항목마다 "직전보다 ○○% 넘게 변하면 알림"을 숫자로 적어 두면, 커지든 작아지든 그만큼 차이가 나면 알려줍니다. 값을 적었을 때만 켜집니다.
  - **알람 전체 스위치를 없앴습니다.** 따로 켜고 끌 필요 없이, 항목에 "증가/감소"나 변동률을 **설정해 두기만 하면 자동으로** 작동합니다.
  - **"조회" 탭을 다시 만들었습니다.** 휴대폰을 세로로 들었을 때 **잘리지 않고** 한눈에 들어오도록 정리했고, 글자 크기가 화면 크기에 맞춰 조절됩니다. **직전 조사와 가장 최근 조사** 두 번만 나란히 비교하며, 샘플마다 값의 변화와 변화율(%)을 봅니다. 여러 샘플을 더하거나 평균 내지 않고 **각 샘플의 변화를 그대로** 보여줍니다. 카드·표(피봇)·묶음(그룹)·퍼센트 보기를 골라 쓸 수 있습니다.
  - **조사 날짜를 "몇 주차" 대신 기간(예: 6/1~6/7)으로** 보여줍니다. 주차 숫자는 직관적이지 않아서, 그 주에 해당하는 날짜 범위로 표시합니다.
  - **샘플키·조사시기 설정이 "조회" 탭으로 옮겨졌습니다.** 설정탭에서는 빠지고, 조회 탭 위쪽 "표시 설정"에서 필요할 때 펼쳐 조정합니다. "오늘 테이블 생성"은 **"입력 테이블 생성"**으로 이름을 바꿨습니다.
- **v0.7.0** (2026-06-12) — 지난 조사와 비교하는 "조회" 탭이 생기고, 음성 클립과 인식 안정성을 한층 다듬었습니다.
  - **"조회" 탭이 생겼습니다.** 오늘 입력한 값을 **직전 조사 때의 값과 나란히** 비교해 봅니다. 샘플(나무·과실)별로 직전값과 현재값, 증감 화살표가 보이고, 회차를 골라 지난 조사들도 살펴볼 수 있습니다. 아침 일찍(자정~9시) 기록한 세션이 "오늘 기록"으로 안 잡히던 문제도 함께 고쳤습니다.
  - **추세 확인 음성 알림을 켤 수 있습니다(설정탭).** 항목별로 "커져야 정상"/"작아져야 정상"을 정해 두면, 입력한 값이 직전 조사보다 거꾸로 갔을 때 음성으로 알려주고 확인을 기다립니다. "확인"(또는 "유지")이라고 하면 그대로 진행하고, 새 값을 말하면 고쳐집니다.
  - **녹음 끝부분이 잘리지 않습니다.** 말이 끝난 뒤 0.5초를 더 담아, 마지막 음절("...점 칠")이 클립에서 끊기던 것을 보강했습니다.
  - **"수정" 후 다시 말한 값의 음성이 유실되던 문제를 고쳤습니다.** 수정 재녹음이 비어 버려 **이전 값 음성이 새 값의 재생버튼에 그대로 남던**(클립과 값이 안 맞던) 문제를 3중으로 차단했습니다 — 다시 안내한 뒤에도 녹음이 확실히 시작되고, 만에 하나 녹음이 비면 "수정 ○○" 발화 음성으로 대신 연결하거나 재생버튼을 정리합니다.
  - **잘못 들은 숫자를 조용히 저장하지 않고 다시 묻는 범위를 넓혔습니다.** "제17.7", "현백 33.3"처럼 숫자 앞뒤에 엉뚱한 말이 섞여 들리면(백 단위가 유실된 정황) 저장하지 않고 다시 묻습니다. "33.3 밀리", "8 프로" 같은 정상적인 단위 말투는 그대로 저장됩니다.
  - **세션 복구 목록에 시간이 표시됩니다.** 같은 날 여러 번 조사했을 때 어떤 세션인지 시간으로 구분해 고를 수 있습니다.

## 기능

- **설정**: Google OAuth 로그인 → 스프레드시트 URL 붙여넣기 → 컬럼 자동 분석 → 데이터형/입력방식 설정 → 오늘 테이블 생성
- **입력**: 항상 켜진 마이크 + TTS 안내 + 자동 행 진행 + Wake Lock (화면 꺼짐 방지)
- **데이터**: IndexedDB 영속화 + Google Sheets 자동 동기화 + CSV 내보내기

## 음성 명령

> v0.4.0부터 **기능당 한 단어**로 통일했습니다. 입력 화면의 **"？명령어"** 버튼으로 전체 목록을 볼 수 있습니다.

| 키워드 | 동작 |
|--------|------|
| 숫자 | 현재 항목에 입력 (한글 수사·아라비아 모두 지원) |
| `수정` (+ 값) | 직전 입력 값 수정 (`178.1 수정`처럼 값 뒤에 붙여도 됨) |
| `다시` | 현재 항목 재입력 (`다시 8.4`처럼 값 동반 가능) |
| `취소` | 현재 인식값 삭제 후 재입력 대기 |
| `유지` | 현재 항목의 값을 그대로 두고 다음으로 진행 (값이 없으면 음성 안내, v0.5.0부터 재입력 중이 아니어도 동작) |
| `스킵` | 현재 행 건너뛰기 |
| `이전행` / `다음행` | 이전·다음 행으로 이동(검토·수정 후 복귀) |
| `일시정지` / `재시작` | 입력 멈춤 / 재개 |
| `종료` | 세션 종료 |

한국어 수사 예시: `삼십오 점 일` → `35.1`, `일점오` → `1.5`, `이천이십육` → `2026`

## 사용법

1. 스마트폰 Chrome에서 배포 URL 접속
2. 홈 화면에 추가 (PWA 설치)
3. 설정 탭 → Google 로그인 → 스프레드시트 URL 붙여넣기
4. 컬럼 카드에서 각 항목의 데이터형/입력방식 조정
5. `오늘 테이블 생성` 클릭
6. 입력 탭 → 이어폰 착용 → `음성 입력 시작`
7. TTS 안내 → 값 음성 입력 → 자동 진행
