## 변경 내역 (Changelog)

> 최근 변경이 맨 위입니다. 어떤 점이 바뀌었는지 쉬운 말로 정리합니다.

- **v0.5.0** (2026-06-10) — 실기기 테스트(v0.4.5) 로그 분석과 사용 요청을 반영했습니다.
  - **"다음"으로 건너뛴 행이 데이터탭에서 사라지지 않습니다.** 빈 행으로 자리를 지키고(행 번호가 주황색으로 표시), 나중에 그 칸을 터치해 채울 수 있습니다.
  - **행 이동이 항상 위에서 아래로만 진행됩니다.** 건너뛴 행이나 이미 완료한 행으로 멋대로 되돌아가지 않습니다(완료한 행으로 자꾸 되돌아가던 버그를 고쳤습니다). 마지막 행까지 입력하면 비어 있는 행 번호를 음성으로 알려주고 자동으로 종료됩니다.
  - **"유지"가 언제든 동작합니다.** 다시 입력하는 중이 아니어도, 현재 칸에 값이 있으면 그대로 두고 다음으로 넘어갑니다. 유지할 값이 없으면 음성으로 알려줍니다.
  - **잘못 들은 값을 조용히 저장하지 않고 다시 묻습니다.** "수정 이백육십육..."이 잘못 쪼개져 엉뚱한 값으로 저장되던 문제, "111 점 에"처럼 소수점 뒷부분을 못 알아들었는데 정수만 저장되던 문제를 고쳤습니다.
  - **음성 클립 앞부분이 잘리지 않습니다.** 클립 앞에 0.5초를 미리 담아, 안내 음성 도중에 말해도 첫 음절이 온전히 녹음됩니다. 다듬기 전 원본 음성도 함께 보관합니다.
  - **설정탭에 "음성 새로고침" 버튼**이 생겼습니다. iPhone 설정에서 안내 음성을 추가한 뒤 누르면 다시 검색하고, 감지된 한국어 음성 개수를 보여줍니다(iOS가 일부 고품질 음성을 웹에 제공하지 않을 수 있다는 안내 포함).
  - **앱을 지웠다 다시 설치해도, 구글 로그인 후 "세션 복구" 버튼 하나로 Drive에 백업된 조사 기록과 음성까지 모두 되찾을 수 있습니다.** (시트 업로드 후 자동 삭제된 세션도 되살아날 수 있는데, 정상 동작이며 시트에 다시 중복 업로드되지는 않습니다. 오늘 이전의 로그는 구버전이라 "구버전 로그 제외"로 표시됩니다.)
  - (내부) 앱 시작·복구·업로드·음성 목록에 대한 진단 기록을 강화했습니다.

## 기능

- **설정**: Google OAuth 로그인 → 스프레드시트 URL 붙여넣기 → 컬럼 자동 분석 → 데이터형/입력방식 설정 → 오늘 테이블 생성
- **입력**: 항상 켜진 마이크 + TTS 안내 + 자동 행 진행 + Wake Lock (화면 꺼짐 방지)
- **데이터**: IndexedDB 영속화 + Google Sheets 자동 동기화 + CSV 내보내기

## 음성 명령

> v0.4.0부터 **기능당 한 단어**로 통일했습니다. 입력 화면의 **"？명령어"** 버튼으로 전체 목록을 볼 수 있습니다.

| 키워드 | 동작 |
|--------|------|
| 숫자 | 현재 항목에 입력 (한글 수사·아라비아 모두 지원) |
| `수정` (+ 값) | 직전 입력 값 수정 (`178.1 수정`처럼 값 뒤에 붙여도 됨) |
| `다시` | 현재 항목 재입력 (`다시 8.4`처럼 값 동반 가능) |
| `취소` | 현재 인식값 삭제 후 재입력 대기 |
| `유지` | 현재 항목의 값을 그대로 두고 다음으로 진행 (값이 없으면 음성 안내, v0.5.0부터 재입력 중이 아니어도 동작) |
| `스킵` | 현재 행 건너뛰기 |
| `이전행` / `다음행` | 이전·다음 행으로 이동(검토·수정 후 복귀) |
| `일시정지` / `재시작` | 입력 멈춤 / 재개 |
| `종료` | 세션 종료 |

한국어 수사 예시: `삼십오 점 일` → `35.1`, `일점오` → `1.5`, `이천이십육` → `2026`

## 사용법

1. 스마트폰 Chrome에서 배포 URL 접속
2. 홈 화면에 추가 (PWA 설치)
3. 설정 탭 → Google 로그인 → 스프레드시트 URL 붙여넣기
4. 컬럼 카드에서 각 항목의 데이터형/입력방식 조정
5. `오늘 테이블 생성` 클릭
6. 입력 탭 → 이어폰 착용 → `음성 입력 시작`
7. TTS 안내 → 값 음성 입력 → 자동 진행
