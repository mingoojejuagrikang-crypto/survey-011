## 변경 내역 (Changelog)

> 최근 변경이 맨 위입니다. 어떤 점이 바뀌었는지 쉬운 말로 정리합니다.

- **v0.35.2** (2026-07-16) — v0.35.1에 이은 **배관 정리 2편**입니다. 화면·동작·소리 모두 이전과 완전히 같고, 달라진 점을 느끼지 못하는 것이 정상입니다.
  - **큰 화면 코드 3개를 작은 부품으로 나눴습니다.** 입력탭·데이터탭·설정탭이 각각 한 파일에 수천 줄씩 담겨 있던 것을 역할별 부품(컴포넌트 38개 + 로직 훅 2개)으로 분리했습니다. 앞으로 고치고 싶은 곳만 정확히 고칠 수 있어, 수정이 다른 기능을 건드릴 위험이 줄어듭니다.
  - **팝업 10곳의 뼈대를 한 벌로 합쳤습니다.** 노치·홈바를 피하는 계산과 어둡게 깔리는 배경이 전부 한 곳에서 관리됩니다(아이폰 화면 잘림 방지 코드가 팝업마다 제각각이던 것을 통일).
  - **로그 기록 형식의 기준서를 만들었습니다.** 로그를 분석하는 도구와 앱이 같은 약속을 코드로 공유해, 앞으로 로그 형식이 어긋날 일이 없습니다(기존 로그와 100% 호환).
  - **"빠른 인식"이 꺼져 있으면 아무 영향이 없음을 자동 테스트로 못박았습니다**(기능 격리 원칙). 자동 테스트는 676개가 되었고, 이중 AI 코드리뷰 2라운드(94.3점, Critical 0)를 통과했습니다.

- **v0.35.1** (2026-07-16) — 겉모습은 그대로 두고 **집 안 배관을 정리한 릴리스**입니다. 쓰던 방식 그대로 쓰시면 되고, 달라진 점을 느끼지 못하는 것이 정상입니다.
  - **안 쓰던 "비교" 화면을 완전히 들어냈습니다.** 예전에 숨겨두기만 했던 화면의 코드(약 2,900줄)를 삭제해 앱이 조금 가벼워졌습니다.
  - **한 폰에서 구글 계정을 바꿔 로그인해도 기록이 섞이지 않습니다.** 예전에는 A 계정으로 쓰다가 B 계정으로 바꾸면, 드물게 B의 조사 기록이 A 이름의 백업 폴더로 들어갈 수 있는 길이 있었습니다. 이제 백업 폴더를 항상 "지금 로그인한 계정"과 대조해 막습니다.
  - **네트워크가 나쁠 때 백업이 영영 멈춰 있던 길목을 막았습니다.** 백업 요청이 30초 안에 응답이 없으면 끊고 다음 기회에 다시 시도합니다.
  - 그 밖에 같은 일을 하는 코드 여러 벌을 한 벌로 합치고, 자동 테스트를 671개로 늘렸습니다(이중 AI 코드리뷰 4라운드 통과). 다음 업데이트를 더 빠르고 안전하게 만들기 위한 준비입니다.

- **v0.35.0** (2026-07-16) — 실기기 점검(7/15)에서 주신 개선요청 7건을 반영했습니다. **핵심은 "폰을 2~3m 떨어뜨려 두고 쓸 때 잘 보이고 잘 들리게"** 입니다. 더불어 값이 사라질 수 있던 길목 여섯 군데를 막았습니다.
  - **인식한 값이 화면 한가운데 크게 뜹니다.** 예전엔 인식 결과가 위쪽 작은 칩에만 반영돼, 멀리서는 제대로 알아들었는지 확인할 수 없었습니다. 이제 값이 저장되는 순간 **굵은 초록색 ✓와 함께 큰 숫자**가 잠깐 뜬 뒤 다음 항목으로 넘어갑니다.
  - **"듣는 중" 글자를 없애고 그 자리에 목소리 파형을 넣었습니다.** 말할 때 파형이 실제로 움직여, 멀리서도 **지금 내 말을 듣고 있는지** 한눈에 보입니다. 마이크가 죽어 있으면 파형도 멈춥니다(가짜로 흔들리지 않습니다).
  - **화면 테두리 불빛을 더 또렷하게** 바꿨습니다. 더 좁은 영역에서 선명하게 천천히 깜빡입니다(정상=초록, 이상치=빨강, 일시정지=노랑).
  - **확인음이 커지고 볼륨을 조절할 수 있습니다.** 설정탭에 볼륨 슬라이더가 생겼습니다(기본값도 예전보다 3배 큽니다). 현장 소음에 맞춰 직접 맞추시면 됩니다.
  - **마지막 행까지 끝내면 아래 버튼이 "종료"로 바뀝니다.** 예전엔 다 끝났는데도 "일시정지" 버튼이 남아 있어 어디를 눌러야 할지 알 수 없었습니다.
  - **설정탭 아래 '설정 요약'이 접혀 있습니다.** 필요할 때 눌러서 펼쳐 보시면 됩니다.
  - **과거값이 미리 준비됐는지 초록색 ✓로 표시합니다.** 로그인하고 시트를 연결하면 과거값을 미리 받아 두므로, **현장에서 인터넷이 안 되어도 이상치 알람이 동작합니다.**
  - **값이 사라질 수 있던 문제들을 막았습니다.** 종료할 때 저장이 끝나기 전에 새 조사를 시작할 수 있던 점, 저장에 실패했는데도 그냥 끝나 버리던 점(이제 실패하면 **빨간 알림이 뜨고 다시 저장할 때까지 종료되지 않습니다**), 종료 확인창이 떠 있는 동안 주변 소리가 값으로 잘못 들어가던 점, 종료하는 중에 버튼이 눌려 방금 입력이 날아가던 점을 모두 고쳤습니다.

- **v0.34.0** (2026-07-15) — 실기기 점검(7/14)에서 주신 개선요청 5건을 반영하고, 손으로 입력한 값이 확인 전에 시트로 새어나가던 문제들을 근본적으로 막았습니다.
  - **손으로 입력한 값이 이상치일 때, 확인하기 전까지는 아무 데도 반영되지 않습니다.** 예전에는 이상치 팝업이 떠 있는 동안에도 값이 이미 저장돼, 그 상태로 데이터탭에서 시트에 올리면 **확인하지 않은 값이 그대로 올라가거나**, 앱을 껐다 켜면 **확인 대기 중이던 값이 확정된 것처럼 남아** 있었습니다. 이제 확인 전 값은 따로 보관되어 시트·CSV·백업 어디에도 나가지 않고, 앱을 껐다 켜도 팝업과 대기 상태가 그대로 복구됩니다.
  - **이상치 팝업이 떠 있는 동안에는 [확인]/[수정] 터치만 받습니다.** 지나가는 트럭 소리나 혼잣말이 숫자로 잘못 인식돼 방금 넣은 값을 덮어쓰거나, [다음]·[이전]·[일시정지]를 눌러 확인 없이 넘어가던 길을 모두 막았습니다.
  - **잘못된 칩이 활성화되던 문제(제보 1)를 고쳤습니다.** 칩을 눌러 손으로 입력했을 때 이상치 팝업이 뜨면 대상 칩이 다음으로 넘어가 엉뚱한 칩을 가리키던 문제입니다.
  - **"이전"으로 돌아갔을 때 그 행의 첫 항목부터 수정할 수 있습니다(제보 3).** 예전엔 포인터가 마지막 값으로 가서 첫 항목은 손으로만 고칠 수 있었습니다. "수정 횡경"처럼 **항목 이름을 지목**해 고칠 수도 있습니다.
  - **화면이 정리됐습니다(제보 2·4).** 일시정지 버튼 위의 작은 "저장됨" 줄과, 완료 후 화면 가운데 뜨던 '입력 완료' 카드를 없앴습니다.
  - **이상치 팝업이 읽기 쉬워졌습니다(제보 5).** 글자를 키우고 "직전값 → 현재값"을 한 줄에 모았습니다.
  - **말한 크기가 화면 테두리에 은은하게 표시됩니다.** 현재 상태(정상 초록 / 이상치 빨강 / 일시정지 노랑)를 테두리 빛으로도 알 수 있습니다.
  - **잘못 알아들은 값이 몰래 저장되는 길을 더 막았습니다.** "예/네" 같은 대답이 숫자 4로 저장되거나, 앞자리가 빠진 채(예: "현백 33.3" → 33.3) 저장되던 경우를 재질문하도록 바꿨습니다.
  - **저장이 실패하면 실패했다고 알립니다.** 예전에는 기기 저장이 실패해도 화면상 성공처럼 보여, 앱을 다시 켜면 값이 사라져 있을 수 있었습니다.
  - (기술) 로그인 없이도 공개 시트에서 과거값을 읽어 알람을 유지합니다. 권한 오류(403)면 무의미한 재시도를 즉시 멈춥니다. 자세한 전제는 `KNOWN-ISSUES.md` [AUTH-10] 참고.

- **v0.33.0** (2026-07-14) — 실사용 제보(마이크 끊김·로그인 풀림·잘못 눌림·화면 가장자리 잘림)와 QA 결과를 한 번에 반영한 대형 업데이트입니다.
  - **다른 탭에 다녀와도 음성 인식이 계속 살아 있습니다(입력탭).** 입력 중 데이터탭·설정탭으로 이동했다 돌아오면 음성 인식이 멈춰 수동으로 일시정지/재개를 해야 했던 문제를 근본적으로 고쳤습니다. 화면을 껐다 켜거나 다른 앱에 다녀온 뒤에도 즉시 스스로 복구를 시도합니다.
  - **Google 로그인이 풀려 있어도 이상치 알람이 계속 울립니다.** 과거 조사값을 기기에 안전하게 보관해 두고(14일), 로그인이 만료된 상태에서도 "직전 조사보다 크게 다른 값"을 놓치지 않고 알려 줍니다. 설정탭·입력 시작 화면에 **Google 로그인 / 시트 연결 / 과거값 준비** 상태가 각각 따로 보여, 알람이 준비됐는지 시작 전에 확인할 수 있습니다.
  - **소수점 되묻기에서 잘못된 값이 저장되던 문제를 고쳤습니다.** 예: "211점..." 후 소수점 아래를 되물었을 때 대답을 잘못 알아들으면 엉뚱하게 "1" 같은 값이 통째로 저장될 수 있었습니다. 이제 되묻는 동안에는 소수점 아래 숫자로만 해석하고, 알아듣지 못하면 다시 물어봅니다.
  - **"이전"과 "다음"이 버튼과 똑같이 단순 이동으로 통일됐습니다.** 이미 완료된 행에 도착하면 저장된 값을 읽어 주고 기다립니다 — "수정"이라고 말하면 고칠 수 있고, "다음"으로 넘어갑니다.
  - **말하기 어려울 때는 항목 칩을 눌러 손으로 입력할 수 있습니다(입력탭).** 항목 종류에 맞는 큰 키패드/선택 버튼이 뜨고, 입력하면 음성과 똑같이 저장·검사됩니다. 시트 안의 "음성으로 다시 입력" 버튼으로 언제든 음성으로 돌아갑니다.
  - **이상치 알람 화면에 [확인]·[수정] 버튼이 생겼습니다(입력탭).** 시끄러워서 말이 안 통할 때 버튼으로도 똑같이 처리할 수 있습니다.
  - **값을 저장할 때마다 화면 아래에 "저장됨 — 몇 행 어떤 항목 값" 영수증이 남습니다(입력탭).** 다음 항목으로 넘어간 뒤에도 방금 저장이 제대로 됐는지 눈으로 확인할 수 있습니다.
  - **데이터탭에서 값과 음성 재생 버튼이 분리됐습니다.** 값을 고치려다 재생 버튼이 눌리던 문제가 구조적으로 사라졌고, 세션 카드에 **작성중 N** 표시가 생겨 아직 덜 끝난 조사를 구분할 수 있습니다.
  - **화면 위아래가 기기 노치/홈바에 가려지던 문제를 고쳤습니다.** 팝업·목록·버튼이 아이폰 노치와 하단 바를 피해 표시됩니다.
  - **개선요청 탭이 새로 생겼습니다.** 쓰다가 불편한 점이 생기면 그 자리에서 탭을 누르세요 — 지금 화면이 자동으로 캡처되고, 한 줄 적어 보내면 화면과 함께 전달됩니다(인터넷이 안 되면 보관했다가 나중에 자동 전송).
  - **설정탭 추가:** ① 도움말에 **항목 종류(날짜/텍스트/정수/실수/리스트/이름) 설명**이 생겼습니다. ② **입력화면 자동 캡처**(기본 켜짐) — 입력 중 화면 변화를 자동 저장해 문제 분석에 사용합니다(로그 내보내기에 포함). ③ **비프음 선택** — 긍정/부정 알림음을 각 5가지 중에서 눌러 들어보고 고를 수 있습니다.

## 기능

- **설정**: Google OAuth 로그인 → 스프레드시트 URL 붙여넣기 → 컬럼 자동 분석 → 데이터형/입력방식 설정 → 오늘 테이블 생성
- **입력**: 항상 켜진 마이크 + TTS 안내 + 자동 행 진행 + Wake Lock (화면 꺼짐 방지)
- **데이터**: IndexedDB 영속화 + Google Sheets 자동 동기화 + CSV 내보내기

## 음성 명령

> v0.4.0부터 **기능당 한 단어**로 통일했습니다. 입력 화면의 **"？명령어"** 버튼으로 전체 목록을 볼 수 있습니다.

| 키워드 | 동작 |
|--------|------|
| 숫자 | 현재 항목에 입력 (한글 수사·아라비아 모두 지원) |
| `수정` (+ 값) | 직전 입력 값 수정 (`178.1 수정`처럼 값 뒤에 붙여도 됨) |
| `다시` | 현재 항목 재입력 (`다시 8.4`처럼 값 동반 가능) |
| `취소` | 현재 인식값 삭제 후 재입력 대기 |
| `유지` | 현재 항목의 값을 그대로 두고 다음으로 진행 (값이 없으면 음성 안내, v0.5.0부터 재입력 중이 아니어도 동작) |
| `스킵` | 현재 행 건너뛰기 |
| `이전행` / `다음행` | 이전·다음 행으로 이동(검토·수정 후 복귀) |
| `일시정지` / `재시작` | 입력 멈춤 / 재개 |
| `종료` | 세션 종료 |

한국어 수사 예시: `삼십오 점 일` → `35.1`, `일점오` → `1.5`, `이천이십육` → `2026`

## 사용법

1. 스마트폰 Chrome에서 배포 URL 접속
2. 홈 화면에 추가 (PWA 설치)
3. 설정 탭 → Google 로그인 → 스프레드시트 URL 붙여넣기
4. 컬럼 카드에서 각 항목의 데이터형/입력방식 조정
5. `오늘 테이블 생성` 클릭
6. 입력 탭 → 이어폰 착용 → `음성 입력 시작`
7. TTS 안내 → 값 음성 입력 → 자동 진행
